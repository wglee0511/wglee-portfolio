## 자기소개

4년 차 개발자로 웹과 앱 서비스를 개발, 배포 및 운영했습니다. 작은 규모의 팀에서 200명 규모의 회사로 성장하는 과정 속에서, 제품 개발에 필요한 기술 역량과 협업 방식을 함께 쌓아왔습니다.

팀의 생산성과 작업 효율성을 높이기 위해 공용 UI 컴포넌트와 재사용 가능한 라이브러리를 제작하고 운영하며, 반복 작업을 줄이고 개발 속도를 개선해왔습니다.

문제를 해결할 때는 혼자 고민하기보다 다양한 역할의 동료들과 적극적으로 소통하는 방식을 선호합니다. 새로운 기술 도입이나 구조 변경이 필요한 경우에는 PoC(Proof of Concept)를 통해 가능성과 효과를 먼저 검증하고, 이를 바탕으로 팀원들과 공감대를 형성하며 설득해왔습니다.

- 코인 소모·획득·힌트 구매 등 모든 경제 액션을 `useMutation + Optimistic Update`로 처리

**제약 충족(Constraint Satisfaction) 알고리즘 기반 AI 플레이어 구현**

- Fisher-Yates 셔플로 첫 추리에 무작위성 부여, 이후 확정 후보 선택으로 **평균 4~5회 내 정답 도달**

- Python 기반 **암호화폐 자동매매 시스템**을 개인적으로 설계·구현한 사이드 프로젝트
- 업비트(Upbit) 거래소 **실계좌와 연동**하여 24시간 무인 자동 운용

#### 주요 구현 내용

**복합 기술적 분석 알고리즘 설계**
- 단일 지표 의존에 따른 과적합 및 오신호 문제를 해결하기 위해 **다중 지표 조합 구조 직접 설계**
- EMA 200 + Stochastic RSI, Bollinger Bands + RSI, MACD, Delta Volume, Fibonacci Retracement, KNN, Lorentzian Distance 등 **9가지 전략 모듈** 구현
- 횡보장 판별 알고리즘을 직접 설계하여 추세 구간에서만 전략이 동작하도록 필터링
- ATR 비율과 볼린저 밴드 폭을 가중 평균(각 0.5)한 동적 기준으로 횡보 여부를 판단

**시장 조건 필터링 및 리스크 관리 시스템**
- 업비트 시장 지수(UBMI/UBAI) API를 통해 전체 시장 변동률을 **실시간 모니터링**
- UBMI 또는 UBAI 변동률이 -2% 미만이면 신규 매수 차단
- -3% 미만 시 보유 포지션 즉시 청산 트리거

**동적 손절/익절 로직 구현**
- 시장 구조 기반의 동적 손절/익절 로직 직접 설계
- ATR(14)과 최근 20봉 전저점을 결합한 동적 손절가 계산: `stop_loss = 최근 20봉 최저가 - (ATR × 1.5)`
- 손절폭을 기준으로 R:R 1:2 비율의 익절가 자동 설정: `take_profit = 매수가 + (매수가 - 손절가) × 2`
- EMA 200 기울기를 선형회귀(numpy.polyfit)로 계산하여 추세 방향을 수치화

**포지션 사이징 자동화**
- 포지션별 리스크를 균등하게 분배하기 위해 동적 포지션 사이징 로직 설계
- 전체 자산 대비 목표 배분 금액을 자동 계산하여 매수 비율로 환산

**멀티스레드 기반 코인 스캔 최적화**
- 거래량 상위 50개 코인을 순차 조회를 위해 **병렬 처리 구조** 도입
- `ThreadPoolExecutor(max_workers=3)`를 활용하여 동시 API 호출로 스캔 속도 개선
- 거래량 70% + 모멘텀 점수 30%의 가중치 기반 종합 점수로 코인 우선순위 결정
- API Rate Limit 대응을 위해 **지수 백오프(0.5초 → 1초 → 2초)** 및 최대 3회 재시도 로직 적용

**CI/CD 파이프라인 구축 및 GCP 배포 자동화**
- GitHub Actions를 활용하여 main 브랜치 push 시 자동 배포 설정
- Docker 이미지를 GCP Artifact Registry에 SHA 태그와 함께 push하고, GCP VM 인스턴스에 SSH 원격 배포

**Slack 기반 실시간 모니터링 알림 시스템**
- 24시간 자동 운용 시스템에서 이벤트 인지 지연을 방지하기 위해 Slack 알림 시스템 직접 연동
- 매수/매도/오류/주문취소 이벤트 발생 시 Slack SDK를 통해 KST 기준 실시간 메시지 발송
- 매 정각마다 계좌 잔고, 코인 평가금액, 전체 수익률, 시장 지수, 보유 코인 상세 정보를 자동 리포팅

**백테스트 프레임워크 구현**
- 신규 전략을 실 거래에 적용하기 전 성능을 검증하기 위해 백테스트 모듈 직접 설계 및 구현
- 업비트 과거 캔들 데이터를 분봉 단위로 페이지네이션 수집하여 임의 기간 시뮬레이션 지원
- 수수료(0.05%) 반영, 진입/청산 시점 기록, 승률/평균 수익/평균 손실/평균 거래 시간 등 성과 지표 자동 산출
- 손절/익절 모드를 상수로 토글하여 전략 조합별 성과 비교 가능

#### 사용 기술
`Python 3.11` `pandas` `numpy` `ta` `scikit-learn` `Docker` `GCP (Compute Engine, Artifact Registry)` `GitHub Actions`
